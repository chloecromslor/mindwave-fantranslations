COMMENT INSTALLER
1. Dans votre bibliothèque Steam, faites un clic droit sur MINDWAVE Demo, puis cliquez sur Gérer>Parcourir les fichiers locaux.
2. Copiez les contenus du patch francais (loc, scripts et data.win) et collez-les dans les fichiers de la démo. 
3. Lancez le jeu.

ATTENTION!
Le jeu est concu pour fonctionner avec un clavier QWERTY et une souris. Si votre clavier utilise le format AZERTY, il vous est fortement recommandé de changer sa configuration.
Le jeu n'est pas concu pour fonctionner avec des manettes, et il est déconseillé de jouer avec le pavé souris.
Certains effets peuvent affecter certaines personnes souffrant de crise d'épilépsies. Conseillé à un public avisé.
